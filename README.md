# City World Flavour

A flavour and content mod for [Stellaris](https://store.steampowered.com/app/281990) that adds depth, specialisation, and life to Ecumenopolis worlds.

[![Steam Workshop](https://img.shields.io/badge/Steam_Workshop-Subscribe-1b2838?logo=steam)](https://steamcommunity.com/sharedfiles/filedetails/?id=3051471269)
[![Discord](https://img.shields.io/badge/Discord-Modding_Den-5865F2?logo=discord&logoColor=white)](https://discord.gg/bHVez2C)

**Stellaris compatibility:** 4.\*

> The legacy 3.14 build is maintained separately as [\[3.14\] City World Flavour](https://steamcommunity.com/sharedfiles/filedetails/?id=3476394622).

---

## Features

### Ecumenopolis
- **Deep Levels** — want to play tall? Expand your Ecumenopolis by adding new levels deep underground. Be careful though; the deeper you go, the harder it becomes to manage your pops down there.
- **Specialisation** — push your planet further into a chosen identity. Turn it into a glittering centre of galactic politics, or a gritty industrial forge world.
- New decisions to turn your Ecumenopolis into the shining jewel of the galaxy it was always meant to be

### Events
- 100+ new flavour events and event chains to add life and character to a bustling intergalactic metropolis

### Jobs
- New jobs specific to Ecumenopolis worlds

## Installation

### From the Steam Workshop (recommended)
[Subscribe on the Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3051471269) — the mod will download automatically and appear in your Stellaris launcher.

### From source
If you'd rather install manually (e.g. to run a development version), clone or download this repository into your Stellaris `mod` folder:

| OS      | Path                                                           |
|---------|----------------------------------------------------------------|
| Windows | `%USERPROFILE%\Documents\Paradox Interactive\Stellaris\mod\`   |
| macOS   | `~/Documents/Paradox Interactive/Stellaris/mod/`               |
| Linux   | `~/.local/share/Paradox Interactive/Stellaris/mod/`            |

You will also need a `.mod` descriptor file in the `mod/` folder pointing at the cloned directory. Then enable **City World Flavour** in the launcher's Mods tab.

> The mod is **savegame compatible**, but any Ecumenopolis that already existed before the mod was added will not have access to all features. Starting fresh — or arcology-projecting a new one — is recommended.

## Compatibility

This mod **does not overwrite any vanilla files**, so it should be compatible with just about anything.

| Mod                                          | Status                                                                                                |
|----------------------------------------------|-------------------------------------------------------------------------------------------------------|
| Planetary Diversity (Arcologies)             | Patch required — [City World Flavour Compatibility Patch](https://steamcommunity.com/sharedfiles/filedetails/?id=3069084845) |
| Real Space (Arcologies)                      | Patch required — [City World Flavour Compatibility Patch](https://steamcommunity.com/sharedfiles/filedetails/?id=3069084845) |
| Sparble's Stellar Hyperconstructs            | Patch required — [Hyperconstructs: City World Flavour Patch](https://steamcommunity.com/sharedfiles/filedetails/?id=3454517304) |

If you find a conflict not listed here, please open an issue or report it on the Workshop page.

## Reporting bugs and contributing

- **Bugs / suggestions:** open an [issue](../../issues) here, or post on the [Workshop comments](https://steamcommunity.com/sharedfiles/filedetails/?id=3051471269) / [Discussions](https://steamcommunity.com/sharedfiles/filedetails/discussions/3051471269)
- **Discussion:** join the [Modding Den Discord](https://discord.gg/bHVez2C)
- **Pull requests:** welcome — please describe what your change does and why. For larger content additions, consider opening an issue first to check it fits the mod's direction.

When reporting a bug, it helps a lot to include:
- The Stellaris version
- A list of other mods you're running (load order matters)
- Steps to reproduce, and a save file if possible
